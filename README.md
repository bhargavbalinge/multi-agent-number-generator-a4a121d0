# Random Number Generator

## Description

This Python script generates a random number between 1 and 6 (inclusive). It can be used to simulate dice rolls or for testing purposes.

## Requirements

- Python 3.x

## Usage

1.  Save the script as `random_number_generator.py`.
2.  Open a terminal or command prompt.
3.  Navigate to the directory where you saved the script.
4.  Run the script using the command: `python random_number_generator.py`

## Output

The script will output a single random number between 1 and 6, like this:

```
Generated number: 4
```
